네트워크는 현실의 고속도로와 비슷하다.

교차로를 만났을 때 경로 선택의 기준은 이정표이다.

네트워크의 교차로 => "Switch"

스위치에서 경로 선택 => 인터페이스를 선택 하는것 = 스위칭

IP주소를 근거로 스위칭  (ip주소,L3계층)
L3 스위칭 

자동차 = "패킷(packet)" 단위
패킷이 인터넷이란 도로를 주행하다 스위치를 다니는데 
스위치 => "라우터"

이정표를 따라 주행 

이정표 => "라우팅 테이블" 

만약 mac주소 스위칭 => L2 스위칭

"라우팅 테이블" = 이정표 이다.


항상 고민해야 하는 주제는 "비용"이다.

도로를 주행할 때 최적 경로를 가야 효율적으로 갈 수 있다.

pc에서 라우팅 테이블 정보 확인하기 

cmd = route => route print 
"메트릭" = 비용
