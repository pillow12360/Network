Host는 컴퓨터인데 네트워크(인터넷)에 연결된 컴퓨터이다.

Host는 두가지로 나뉜다.

Switch 는 Network 그 자체를 이루는 Host
ip주소가 설정되면 다 호스트

네트워크 자체를 확장하거나 기능요소이다. => Switch (인트라)

End-Point => 이용주체

네트워크 인프라를 사용하는 주체

이것을 한국말로 번역하면 : 단말기 (End-point)

단말기의 대표적인 것 

Client : 받음

Server : 줌

Peer : 일방적으로 서비스하건 제공 받는 것이 아닌, 둘다 기능
